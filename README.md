# 🏏 IPL Data Analysis (2008-2019)

## Overview
Exploratory data analysis on 12 seasons of IPL match data using Python, Pandas, and Matplotlib/Seaborn.

## Dataset
- **Source:** Kaggle IPL Dataset
- **Matches:** 752 clean matches across 12 seasons
- **Files:** matches.csv, deliveries.csv

## Tools Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Jupyter Notebook

## Key Insights
1. **Mumbai Indians** are the most successful IPL team with **109 wins**
2. **Winning toss gives only 52% advantage** — toss barely impacts match result
3. **61% of teams choose to field first** after winning toss — chasing is preferred
4. **2013 was the busiest season** with 76 matches played
5. **Eden Gardens** is the most used venue with 77 matches hosted
6. **CH Gayle** is the top performer with 21 Player of the Match awards
